# Non-Invasive Glucose Monitoring

This repository presents the embedded acquisition system for a non-invasive glucose sensing approach based on near-infrared (NIR) spectroscopy.

The implementation focuses on multi-sensor data acquisition using MAX30102 devices, leveraging redundant sampling strategies to improve signal reliability.

# System Overview
The system captures optical signals (RED & IR) from biological tissue and stores structured data for further analysis.

## Key characteristics:

- Multi-channel acquisition (4 sensors)
- I2C multiplexing architecture
- High-frequency sampling
- Real-time data logging in CSV format

# Architecture
MAX30102 Sensors (x4) → I2C Multiplexer (TCA9548A) → Raspberry Pi Pico (MicroPython) → Data Acquisition → CSV Output

# Implementation Details
## Acquisition Strategy
- Sequential channel switching via I2C
- Sensor reset per channel to avoid interference
- FIFO-based reading (RED & IR signals)
- Configurable sampling parameters

# Code Structure

# Design Decisions

- Redundant sensing: improves robustness
- Channel isolation: reduces cross-noise
- Lightweight implementation: optimized for MicroPython
